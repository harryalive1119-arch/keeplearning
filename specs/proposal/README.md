# 悬疑侦探游戏 Demo - 规范文档目录

## 概述
这是一个悬疑侦探类游戏的最小可玩demo，采用2D侧视角视觉小说风格，玩家在单一犯罪现场房间中自由移动、对话、收集证物，最后通过逆转裁判式的问答系统揭晓谜底。

## 文档结构
1. `core_game_loop.md` - 核心游戏循环与状态机
2. `movement_dialogue.md` - 移动与对话系统
3. `evidence_collection.md` - 证物收集系统
4. `truth_revelation.md` - 谜底揭晓系统
5. `data_models.md` - 数据模型与资源定义

## 设计原则
- **玩家体验**：沉浸式侦探体验，强调逻辑推理与证据链构建
- **交互反馈**：即时视觉/音频反馈，参考《逆转裁判》的证物出示系统
- **数据驱动**：所有游戏内容（对话、证物、问题）通过资源文件配置
- **模块化**：各系统通过信号松耦合，便于扩展
